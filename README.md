# 📊 Stress–Strain Curve Analysis

## 📌 Overview

This project focuses on the analysis of **stress–strain behavior** of materials under different loading conditions. It includes computational and graphical evaluation of both:

* **Tensile Test**
* **Compression Test**

The objective is to understand material properties such as elasticity, plasticity, and failure characteristics using real experimental or sample data.

---

## ⚙️ Features

* 📈 Stress–strain curve generation
* 🔍 Analysis of tensile and compressive behavior
* 📊 Data processing from Excel files
* 🧮 Calculation of stress and strain values
* 📉 Visualization using plots

## 🧠 Concepts Used

* Stress = Force / Area
* Strain = Change in Length / Original Length
* Hooke’s Law
* Elastic and Plastic Deformation
* Yield Strength and Ultimate Strength

---

## ▶️ How to Run

1. Install required libraries:

```
pip install numpy pandas matplotlib
```

2. Run tensile test analysis:

```
python python_code_for_tensile_test.py
```

3. Run compression test analysis:

```
python python_code_for_compression_test.py
```
